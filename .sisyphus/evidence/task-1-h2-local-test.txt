## 4. 环境备注（Gotchas）

- **Docker Desktop ryuk 500**：本机（macOS Docker Desktop 28.4.0）Testcontainers 的 ryuk 容器启动 500。绕过方式：`export TESTCONTAINERS_RYUK_DISABLED=true`。这是环境特性，不影响 IT 代码正确性；CI 环境通常无此问题。
- **git hooksPath**：`core.hooksPath = frontend/.husky/_`（husky 前端钩子），后端 `.githooks/pre-commit` 未激活，故本次提交不触发后端 spotless 钩子（但 `mvn test` 的 validate 阶段已跑 spotless:check + checkstyle 并通过）。
- **H2 版本提示**：Flyway 警告 `H2 2.2.224 newer than tested 2.2.220`，仅为提示，不影响迁移成功。
- **Hibernate dialect**：`H2Dialect does not need to be specified explicitly`（HHH90000025）——显式指定 dialect 仍可工作，仅为告警。
